# How to Get Your Gemini API Key

## Step 1: Visit Google AI Studio

Go to: **https://makersuite.google.com/app/apikey**

## Step 2: Sign In

- Sign in with your Google account
- If you don't have one, create a free Google account

## Step 3: Create API Key

1. Click on "Create API Key" button
2. Select "Create API key in new project" (or choose existing project)
3. Copy the generated API key

## Step 4: Add to Website Builder

1. In the dashboard, click "Enable AI Generation"
2. Paste your API key
3. Click "Save"
4. You're ready to generate AI-powered websites!

## Important Notes

### Free Tier Limits
- 60 requests per minute
- Perfect for personal use and testing
- No credit card required

### Security
- Your API key is stored locally in your browser
- Never share your API key publicly
- You can remove it anytime from the dashboard

### API Key Format
Your API key will look like: `AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`

## Troubleshooting

### "Failed to generate website"
- Check if your API key is correct
- Ensure you have internet connection
- Try again in a few seconds

### "API key not provided"
- Click "Enable AI Generation"
- Enter your API key
- Click "Save"

### Rate Limit Exceeded
- Wait 60 seconds
- Free tier: 60 requests/minute
- Consider upgrading if needed
